Welcome to the midiana.lv

My name is Artur Klesun, and this is my web-site. Here you can find the app-s I wrote: links, docs, web-interface and the data they require or produce.

Please, choose the app you are interested in:

  • deep-assoc-completion (src) a phpstorm plugin for associative array auto-completion inferred from other functions
  • deep-js-completion (src) a webstorm plugin for object property auto-completion inferred from other functions
  • deep-dict-completion (src) a pycharm plugin for dict auto-completion inferred from other functions
  • midiana (src) app to enter notes from a MIDI device and edit them
  • fix-web-sites (src) Chrome plugin to improve some site user experience in different ways (filter facebook posts/sort youtube comments/blacklist sites in google/etc...)
  • Starve Game (src) a browser word game
  • MAL DB (src) a DB dump from myanimelist.net, allows you to find anime you would like with advanced search criteria and different average score algorithm
  • Soundfont Player (src) a JS soundfount player implementation and a MIDI player with 3000+ MIDI rated songs from my collection
  • denisbook (src) A work-in-progress open social network. Because facebook is unusable and vk does not have Durov anymore.
  • text_transform A set of useful functions to process text data formats in a GUI. Stuff like conversion between php var_export, json, csv, http query, MySQL SELECT output, etc...
